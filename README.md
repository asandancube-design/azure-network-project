# azure-network-project

## Overview
This project demonstrates my hands-on experience with cloud networking using Microsoft Azure. I designed and deployed a Virtual Network (IPv4) to understand how cloud-based network infrastructure is structured and secured.

## Objectives
- Build a foundational understanding of cloud networking
- Practice IP addressing and CIDR notation
- Learn how subnetting is used for network segmentation in Azure

## What I Did
- Created a Virtual Network (VNet) in Microsoft Azure
- Configured a custom IPv4 address space
- Designed and implemented subnets within the VNet
- Explored how Azure organizes and  isolates network resources

## Key Concepts Learned
- Virtual Networking in cloud environments
- CIDR notation and IP address planning 
- Network segmentation and isolation
- Basics of infrastructure design

## Tools Used
- Microsoft Azure (Virtual Network)
- Azure Portal

## Security Relevance
This project helped me understand how network boundaries are created in cloud environments, which is essential for roles in cybersecurity such as SOC analysis, where monitoring and controlling network traffing is critical.
